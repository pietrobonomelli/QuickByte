## Come eseguire l'applicazione QuickByte:
#### 1. Clonare la directory 
```

```
#### 2. Spostarsi nella directory del progetto
```
cd QuickByte\QuickByte
```
#### 3. Successivamente esegurie il :
```
mvn clean javafx:run
```
